# akashchandru.me — Personal Portfolio

My personal portfolio site built with **React + Vite**, showcasing my work, experience, and skills.

🌐 **Live:** [akashchandru.me](https://akashchandru.me)

## Features

- Projects, experience, skills, and certifications
- AI chatbot integration
- Responsive design with smooth animations

## Tech Stack

React · Vite · CSS Modules

## Run Locally

```bash
npm install
npm run dev
```
